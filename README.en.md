# ImagePlayer

A feature-rich image player application that provides a smooth image browsing experience and various practical functions.

## Key Features

- Supports viewing of multiple image formats
- Provides basic operations such as image zooming and rotating
- Supports full-screen mode for image viewing
- Includes image information viewing capability
- Supports hardware-accelerated rendering
- Integrated OpenCL-based image blur processing
- Provides system tray support
- Supports proxy server settings
- Automatically checks for updates
- Multi-language support (including Simplified Chinese, Traditional Chinese, and English)

## System Requirements

- Windows 7 or later
- Requires Java Runtime Environment
- OpenCL-capable GPU device (optional)

## Installation Instructions

1. Download the installation package
2. Run the installer
3. Complete the installation using the installation wizard
4. Launch the application

## Usage Instructions

1. After launching the application, load images using the file selector
2. Use toolbar buttons to perform zooming, rotating, and other operations
3. Right-click on an image to open the context menu for additional operations
4. Adjust display modes, proxy server, and other options in the settings interface
5. Supports drag-and-drop operations for loading images

## Developer Information

This project is developed using a combination of Java and C++, with a modular design:

- Java components are built using Gradle
- C++ is used to implement core runtime functionality
- High-performance image processing is achieved using OpenCL
- Utilizes a multi-threaded architecture to enhance performance
- Supports internationalization and localization
